vcdw-bashrc
===========

The .bashrc file for VCDW's CoreOS

## Shortcut commands for VCDW

Run a VCDW container named vcdw
```
$ vcdw
```

Run a VCDW container and add a name
```
$ vcdwrun <mycontainer>
```

Stop all running containers
```
$ vcdwstopall
```

Force remove all containers
```
$ vcdwrmall
```
